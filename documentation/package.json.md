# Структура файла `package.json`

Файл `package.json` — это основной конфигурационный файл в проектах на Node.js, включая проекты на Vue. Он выполняет несколько ключевых функций: управление зависимостями, определение скриптов, хранение метаинформации и настройка инструментов.

## Основные функции
- **Управление зависимостями**: Хранит список библиотек и их версий, разделённых на `dependencies` (для продакшена) и `devDependencies` (для разработки).
- **Скрипты**: Определяет команды для автоматизации задач (разработка, сборка, тестирование).
- **Метаинформация**: Содержит данные о проекте (название, версия, автор).
- **Конфигурация**: Настраивает инструменты, такие как ESLint, TypeScript или Vite.

## Разбор `package.json`

```json
{
  "name": "vue-documentation-playground",
  "version": "0.0.0",
  "private": true,
  "type": "module",
  "scripts": {
    "dev": "vite",
    "build": "run-p type-check \"build-only {@}\" --",
    "preview": "vite preview",
    "test:unit": "vitest",
    "build-only": "vite build",
    "type-check": "vue-tsc --build",
    "lint": "eslint . --fix",
    "format": "prettier --write src/"
  },
  "dependencies": {
    "vue": "^3.5.13"
  },
  "devDependencies": {
    "@tsconfig/node22": "^22.0.1",
    "@types/jsdom": "^21.1.7",
    "@types/node": "^22.14.0",
    "@vitejs/plugin-vue": "^5.2.3",
    "@vitest/eslint-plugin": "^1.1.39",
    "@vue/eslint-config-prettier": "^10.2.0",
    "@vue/eslint-config-typescript": "^14.5.0",
    "@vue/test-utils": "^2.4.6",
    "@vue/tsconfig": "^0.7.0",
    "eslint": "^9.22.0",
    "eslint-plugin-vue": "~10.0.0",
    "jiti": "^2.4.2",
    "jsdom": "^26.0.0",
    "npm-run-all2": "^7.0.2",
    "prettier": "3.5.3",
    "typescript": "~5.8.0",
    "vite": "^6.2.4",
    "vite-plugin-vue-devtools": "^7.7.2",
    "vitest": "^3.1.1",
    "vue-tsc": "^2.2.8"
  }
}
```

### Поля файла

- **`"name": "vue-documentation-playground"`**  
  Название проекта. Уникальный идентификатор, используемый для идентификации в npm (если проект публикуется) или для удобства. В данном случае — имя локального проекта.

- **`"version": "0.0.0"`**  
  Версия проекта, следующая семантическому версионированию (SemVer: major.minor.patch). `0.0.0` — начальная версия для проектов в разработке.

- **`"private": true"`**  
  Указывает, что проект частный и не будет опубликован в npm. Предотвращает случайную публикацию, делая проект локальным или внутренним.

- **`"type": "module"`**  
  Указывает, что JavaScript-файлы используют ES Modules (ESM) с синтаксисом `import`/`export` вместо CommonJS (`require`/`module.exports`). Необходимо для современных инструментов, таких как Vite.

- **`"scripts": { ... }`**  
  Объект с командами, запускаемыми через `npm run <script>` или `yarn <script>`.  
  - **`"dev": "vite"`**  
    Запускает Vite в режиме разработки с горячей перезагрузкой (HMR).  
  - **`"build": "run-p type-check \"build-only {@}\" --"`**  
    Выполняет параллельно (с помощью `npm-run-all2`) команды `type-check` (проверка типов TypeScript) и `build-only` (сборка проекта). `{@}` передаёт аргументы.  
  - **`"preview": "vite preview"`**  
    Запускает локальный сервер для просмотра собранного приложения (после `build`).  
  - **`"test:unit": "vitest"`**  
    Запускает юнит-тесты с помощью Vitest, тестового фреймворка, совместимого с Vite.  
  - **`"build-only": "vite build"`**  
    Выполняет сборку проекта для продакшена с помощью Vite (оптимизированный билд).  
  - **`"type-check": "vue-tsc --build"`**  
    Проверяет типы TypeScript с помощью `vue-tsc` (компилятор TypeScript для Vue).  
  - **`"lint": "eslint . --fix"`**  
    Запускает ESLint для проверки и автоматического исправления ошибок кода во всех файлах.  
  - **`"format": "prettier --write src/"`**  
    Форматирует код в папке `src/` с помощью Prettier, переписывая файлы согласно правилам форматирования.

- **`"dependencies": { ... }`**  
  Зависимости, необходимые для работы приложения в продакшене.  
  - **`"vue": "^3.5.13"`**  
    Основной фреймворк Vue.js версии 3.5.13 или новее (до следующей мажорной версии). Используется для создания компонентов и управления UI.

- **`"devDependencies": { ... }`**  
  Зависимости, используемые только в разработке, тестировании и сборке.  
  - **`"@tsconfig/node22": "^22.0.1"`**  
    Преднастроенный конфиг TypeScript для Node.js 22, упрощает настройку `tsconfig.json`.  
  - **`"@types/jsdom": "^21.1.7"`**  
    Типы TypeScript для `jsdom`, используемого для симуляции DOM в тестах.  
  - **`"@types/node": "^22.14.0"`**  
    Типы TypeScript для Node.js, обеспечивают автодополнение и проверку типов для встроенных модулей Node.  
  - **`"@vitejs/plugin-vue": "^5.2.3"`**  
    Плагин Vite для поддержки `.vue`-файлов, необходим для компиляции однофайловых компонентов.  
  - **`"@vitest/eslint-plugin": "^1.1.39"`**  
    Плагин ESLint для интеграции с Vitest, проверяет тестовый код.  
  - **`"@vue/eslint-config-prettier": "^10.2.0"`**  
    Конфигурация ESLint для совместимости с Prettier, предотвращает конфликты линтинга и форматирования.  
  - **`"@vue/eslint-config-typescript": "^14.5.0"`**  
    Конфигурация ESLint для TypeScript в Vue, добавляет правила для TypeScript-кода.  
  - **`"@vue/test-utils": "^2.4.6"`**  
    Утилиты для тестирования Vue-компонентов, используются с Vitest для юнит-тестов.  
  - **`"@vue/tsconfig": "^0.7.0"`**  
    Базовая конфигурация `tsconfig.json` для Vue-проектов, упрощает настройку TypeScript.  
  - **`"eslint": "^9.22.0"`**  
    Линтер для проверки качества кода (JavaScript/TypeScript/Vue), поддерживает единый стиль.  
  - **`"eslint-plugin-vue": "~10.0.0"`**  
    Плагин ESLint для поддержки правил, специфичных для Vue (например, проверка `.vue`-файлов).  
  - **`"jiti": "^2.4.2"`**  
    Библиотека для динамической загрузки ES Modules, поддерживает ESM в инструментах.  
  - **`"jsdom": "^26.0.0"`**  
    Эмулятор DOM для тестов в Node.js, имитирует браузерную среду.  
  - **`"npm-run-all2": "^7.0.2"`**  
    Утилита для параллельного или последовательного выполнения npm-скриптов (используется в `build`).  
  - **`"prettier": "3.5.3"`**  
    Инструмент для автоматического форматирования кода, обеспечивает единый стиль.  
  - **`"typescript": "~5.8.0"`**  
    TypeScript для статической типизации, используется с `vue-tsc`.  
  - **`"vite": "^6.2.4"`**  
    Инструмент сборки и разработки, заменяет Webpack в современных Vue-проектах.  
  - **`"vite-plugin-vue-devtools": "^7.7.2"`**  
    Плагин Vite для интеграции с Vue Devtools, улучшает отладку компонентов.  
  - **`"vitest": "^3.1.1"`**  
    Тестовый фреймворк, оптимизированный для Vite, используется для юнит-тестов.  
  - **`"vue-tsc": "^2.2.8"`**  
    Компилятор TypeScript для Vue, проверяет типы и генерирует декларации для `.vue`-файлов.
